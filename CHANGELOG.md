# Changelog

### Изменено
- Переписана логика инициализации: подписки на события `model_select`, `resources_discover`, `session_start`, `tool_execution_end`, `agent_end`

- Регистрация провайдера pi с поддержкой OpenAI-compatible API
- Провайдер не появляется в списке моделей без установленного `POLZA_AI_API_KEY`
